**Lara Elite** é uma interface de inteligência artificial de alta performance, projetada para oferecer uma experiência fluida, estética e tecnologicamente avançada. O projeto combina o poder do motor de busca Gemini com uma arquitetura PWA (Progressive Web App) de última geração.

## 💎 DESIGN & IDENTIDADE
O design, a interface e a concepção deste projeto foram criados exclusivamente por **Talisson Francisco**. A Lara Elite v30.8 foca em:

* **Estética Cyber-Gold:** Uma UI limpa, luxuosa e responsiva.
* **Experiência Imersiva:** Backgrounds dinâmicos de alta qualidade (Manga/Anime Aesthetic).
* **Foco no Usuário:** Interface projetada para mestre, com controle total de perfil e chat.
* **Log de Sistema Integrado:** Painel de novidades direto no menu lateral para acompanhamento de evolução em tempo real.

## ⚡ FUNCIONALIDADES OMEGA (v30.8)
* **Purga de Cache Atômica:** O Service Worker v30.8 elimina automaticamente fragmentos de versões antigas para garantir performance máxima (Protocolo Buster).
* **Log de Sincronia:** Visualização direta no Cockpit (Menu) de todas as melhorias da versão atual.
* **Omega Search 2026:** Incursão em tempo real na rede para respostas atualizadas e precisas.
* **Modo Projeção (PiP):** Suporte nativo para Picture-in-Picture, permitindo que a Lara flutue sobre outros apps no Android.
* **Notificações de Elite:** Canal de voz e alertas sincronizados diretamente com o sistema.

## 🛠️ TECNOLOGIAS UTILIZADAS
* **Linguagens:** HTML5, CSS3 (Modern Flex/Grid), JavaScript (ES6+).
* **Arquitetura:** PWA com Service Worker v30.8 (Stale-While-Revalidate + Cache Buster).
* **IA:** Pollinations.ai & OpenAI Integration.
* **Markdown:** Renderização via `marked.js`.

## 🚀 COMO EXECUTAR
1.  Hospede os arquivos em um servidor com suporte a HTTPS (necessário para PWA/Service Worker).
2.  Certifique-se de que o `sw.js` e o `index.html` estejam na mesma pasta (raiz).
3.  Acesse via navegador e instale como Aplicativo (Adicionar à tela de início).

## 🤝 APOIO AO PROJETO
A Lara Elite é um projeto independente. Se você deseja apoiar o desenvolvimento e a manutenção dos servidores, sinta-se à vontade para realizar uma doação voluntária via **PIX**. A chave está disponível no menu lateral do aplicativo.
